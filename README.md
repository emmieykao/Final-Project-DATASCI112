### Food Deserts and Swamps - DATASCI 112 Final Project
## File Documentation
`Dataset_genearation.ipynb`: Our initial data collection and cleaning. We begin with two relatively large datasets, select relevant rows, and merge on county `fips` values.
`prelim_analysis.ipynb`: First analysis on individual `x` and `y` values. We analyze relationships between each independent and dependent variable, hoping to find some initial 
correlation to guide future investigation. Interestingly, `income` seems to be the most defining variable, particularly in relation to `obesity`.
